## 评估维度

### 1. 概念理解维度
**评估要点**：
- 是否理解核心概念的定义
- 是否能区分相似概念
- 是否理解概念的适用范围

**判断标准**：
- ✅ 已掌握：能用自己的话准确解释概念，能举出反例
- ⚠️ 部分掌握：能背诵定义，但用自己的话解释时有偏差
- ❌ 未掌握：无法解释或解释完全错误

### 2. 应用能力维度
**评估要点**：
- 是否能举出实际案例
- 是否能将概念应用到新场景
- 是否能设计解决方案

**判断标准**：
- ✅ 已掌握：能举出多个跨领域的实际案例，能设计具体应用方案
- ⚠️ 部分掌握：能举出书本案例，但难以延伸到新场景
- ❌ 未掌握：无法举出实际案例

### 3. 批判思维维度
**评估要点**：
- 是否能识别概念的局限性
- 是否能进行反驳或质疑
- 是否能从反面思考问题

**判断标准**：
- ✅ 已掌握：能提出有价值的反驳观点，能识别适用边界
- ⚠️ 部分掌握：能提出表面质疑，但缺乏深度
- ❌ 未掌握：全盘接受，无法提出质疑

### 4. 知识整合维度
**评估要点**：
- 是否能关联其他知识点
- 是否能构建知识网络
- 是否能形成系统性认知

**判断标准**：
- ✅ 已掌握：能主动关联相关概念，能说明概念间的关系
- ⚠️ 部分掌握：需要提示才能关联其他知识
- ❌ 未掌握：知识点孤立，无法建立联系

## 掌握等级划分

### 等级1：初步了解
**特征**：
- 知道概念的存在
- 能识别概念的基本特征
- 无法独立应用

**学习建议**：继续深入阅读，完成基础练习

### 等级2：基本理解
**特征**：
- 能用自己的话解释概念
- 能举出标准案例
- 应用能力有限

**学习建议**：增加应用场景练习，尝试举一反三

### 等级3：熟练掌握
**特征**：
- 能准确解释概念和区分相似概念
- 能举出多个领域的实际案例
- 能进行基本的批判性思考

**学习建议**：尝试挑战反驳性问题，探索概念边界

### 等级4：深度精通
**特征**：
- 能举一反三，跨领域应用
- 能提出有深度的质疑和反驳
- 能主动构建知识网络

**状态**：达到归档标准

## 评估方法

### 方法1：回答分析
通过分析用户对三类问题的回答质量：

**概念理解题分析**：
- 关键词覆盖：是否提及关键概念词汇
- 逻辑准确性：解释是否自洽
- 表达清晰度：能否用简洁语言说明

**应用题分析**：
- 案例真实性：案例是否来自真实生活
- 应用准确性：应用是否恰当
- 创造性：案例是否具有独特性

**延伸思考题分析**：
- 批判深度：质疑是否有价值
- 论证合理性：论证是否充分
- 视角多样性：是否从多角度思考

### 方法2：迭代观察
通过多轮学习的表现趋势：

- 回答质量是否逐步提升
- 是否能纠正前一轮的理解偏差
- 是否能主动关联前文内容

### 方法3：综合判断
综合所有维度的表现：

```
掌握度 = (概念理解 × 0.3) + (应用能力 × 0.3) + (批判思维 × 0.2) + (知识整合 × 0.2)
```

## 归档条件

### 必要条件（必须全部满足）

1. **概念理解达标**
   - 对所有核心概念的理解等级 ≥ 3
   - 无重大理解偏差

2. **应用能力达标**
   - 能举出至少3个不同领域的实际案例
   - 能设计至少1个具体应用方案

3. **批判思维达标**
   - 能提出有价值的质疑或反驳
   - 能识别概念的适用边界

4. **知识整合达标**
   - 能主动关联至少2个相关知识点
   - 能说明概念间的关系

### 充分条件（满足任一即可）

1. 连续3篇文章的回答质量达到等级4
2. 用户主动表示"我已经完全理解了"
3. 能够教授他人该知识（用教学检验掌握）

### 归档操作

当满足归档条件时，智能体应：

1. **生成学习总结**
   - 核心知识点清单
   - 学习路径回顾
   - 关键感悟摘录

2. **建立知识链接**
   - 识别相关课题
   - 创建双向链接
   - 更新知识网络

3. **输出最终笔记**
   - 使用标准笔记模板
   - 包含概念定义、案例、思考
   - 预留扩展空间

4. **更新进度状态**
   - 将状态标记为"已掌握"
   - 记录完成时间
   - 归档到学习历史

## 使用指导

### 智能体评估流程

1. **阅读用户回答**：使用read_file工具读取用户反馈文件
2. **逐题分析**：根据上述标准分析每个问题的回答质量
3. **维度评分**：对四个维度分别给出评估结果
4. **综合判断**：计算整体掌握度
5. **决策下一步**：
   - 未达标 → 生成针对性文章继续学习
   - 已达标 → 执行归档操作

### 评估语言建议

在给用户反馈时使用：

**鼓励性表达**：
- "你对XX概念的理解很深入..."
- "你的案例很有创意..."
- "你的质疑很有价值..."

**建设性反馈**：
- "在XX方面，建议进一步思考..."
- "可以尝试从XX角度理解..."
- "推荐阅读XX相关内容..."

**归档确认**：
- "恭喜你已经掌握了XX的核心内容！"
- "以下是你的学习总结..."
- "建议将这些知识与XX关联..."
